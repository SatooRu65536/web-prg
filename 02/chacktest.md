## p.47
### Q1
> 数値の 1000 が入った変数名 number を宣言するコードを書いてください。

### 解答
```js
let number = 1000;
```

### Q2
> 文字列の JavaScript が入った変数名 lang を宣言するコードを書いてください。

### 解答
```js
let lang = 'JavaScript';
```

### Q3
> let と const の違いを説明してください。

### 解答
let は変数を、const は定数を宣言するときに使う。
ともに再宣言はできないが、let は再代入できるのに対し、const は再代入できない。
